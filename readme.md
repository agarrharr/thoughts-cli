# thoughts-cli

> Add and get thoughts

## Install

```
$ npm install --global thoughts-cli
```

## Usage

```
$ thoughts --help
  Examples
    $ thoughts add "Would transformers have car insurance? Or health insurance?"
    $ thoughts
    What if Santa is real, but there are no nice children?
```
